# Resource Hiring App – Salesforce Project

## Overview

Resource Hiring App is a Salesforce-based CRM application designed for recruitment, resource allocation, and project workflow management.

This project was developed to improve hiring workflows, manage employee resources, and automate business processes using Salesforce technologies.

---

## Features

* Candidate Management
* Resource Allocation
* Recruitment Workflow Automation
* Project Assignment Tracking
* CRM-Based Management System
* Responsive User Interface

---

## Tech Stack

* Salesforce CRM
* Apex
* SOQL
* Lightning Web Components (LWC)
* HTML
* CSS
* JavaScript

---

## Responsibilities

* Developed frontend UI using LWC
* Implemented backend logic using Apex
* Used SOQL for database operations
* Worked on workflow automation
* Improved CRM functionality
